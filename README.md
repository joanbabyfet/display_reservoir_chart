## About
采集水库数据，存储csv文件且展示成长条图

## Feature

* 采集水库数据, 网站以 https://water.taiwanstat.com 为例
* 通过 matplotlib 库, 实现数据可视化, 并支持导出功能
* 存储数据为csv文件

## Requires
Python 3.11.0  
pandas 1.5.2  
requests 2.28.1  
matplotlib 3.6.2  
BeautifulSoup 4.11.1  

## Usage
```
python main.py
```
